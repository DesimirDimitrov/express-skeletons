# Express-skeletons

Basic skeletons for simple apps (including some legacy)

## List

- npm init

- nodemon

```sh
npm install --save-dev nodemon
```

- dotenv

```sh
npm install dotenv --save
```

- axios

```sh
npm install axios
```

- eslint (with TS settings allow js - true)

```sh
npm init @eslint/config
```
